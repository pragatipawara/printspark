📌 Project Overview

PrintSpark is an AI-powered web application that helps users discover creative 3D printing ideas instantly.

Users simply describe a hobby, interest, passion, or everyday problem, and the application generates unique 3D-printable object ideas along with useful details such as difficulty level, estimated print time, and recommended filament type.

The goal was to create a tool that removes the "What should I print next?" problem faced by many 3D printing enthusiasts.

🎯 Problem Statement

Many people own 3D printers but struggle to come up with practical and creative ideas for their next project.

Finding inspiration often requires browsing multiple websites, communities, and repositories, which can be time-consuming and overwhelming.

This creates a gap between owning a 3D printer and consistently using it to build meaningful projects.

💡 Solution

PrintSpark uses Artificial Intelligence to generate personalized 3D printing ideas based on user input.

Instead of searching manually, users simply describe:

A hobby
An interest
A challenge
A daily-life problem

The system then instantly suggests three unique 3D-printable solutions tailored to that context.

🏗️ How It Works
Step 1

User enters a hobby, interest, or problem.

Step 2

The application sends the prompt to Claude AI.

Step 3

Claude generates structured JSON output containing:

Object name
Emoji representation
Use case
Difficulty level
Print time estimate
Filament recommendation
Step 4

The application displays the generated ideas in an interactive card-based interface.
